Section 01

Why Prediction Sets?

A classifier outputs a single predicted class. In practice, models are often miscalibrated — a 95% softmax score does not mean 95% accuracy. Conformal prediction constructs prediction sets: subsets of classes guaranteed to contain the true label.

ApproachOutputCoverage guaranteed?Calibration-free?
Argmax (standard)Single class❌ No—
Softmax thresholdTop-k classes❌ No❌ No
Temperature scalingSingle class❌ Approximately❌ Needs calibration set
Conformal CPPrediction set✅ Exact finite-sample✅ Any model
💡
Uncertainty through set size

When the model is confident, the prediction set is a singleton {class}. When uncertain, multiple classes are included — giving an honest signal of ambiguity rather than a false single-class answer.

Section 02

Split Conformal for Classification

The algorithm is the same 5-step procedure as regression, but the score function and prediction set construction differ.

Train any classifier f̂ on training data

f̂ should output class probabilities: f̂(x) ∈ Δ^K (probability simplex over K classes).

Compute calibration scores

For each calibration example (xᵢ, yᵢ): sᵢ = 1 − f̂(xᵢ)[yᵢ], i.e., 1 minus the softmax probability assigned to the true class.

Find quantile threshold q̂

q̂ = Quantile({s₁…sₙ}, level = ⌈(n+1)(1−α)⌉/n). Same formula as regression.

Form prediction set at test time

C(x*) = { k : 1 − f̂(x*)[k] ≤ q̂ } = { k : f̂(x*)[k] ≥ 1 − q̂ }. Include all classes whose softmax score is above threshold.

C(x*) = { k ∈ {1…K} : f̂(x*)[k] ≥ 1 − q̂ }
Section 03

Basic CP: Softmax Score

scikit-learnnumpy
basic_conformal_classification.py
import numpy as np
from sklearn.datasets import make_classification
from sklearn.neural_network import MLPClassifier
from sklearn.preprocessing import StandardScaler
from sklearn.model_selection import train_test_split

# ── 5-class synthetic dataset ─────────────────────────────────────
X, y = make_classification(
    n_samples=3000, n_features=20, n_classes=5,
    n_informative=12, random_state=42)
X = StandardScaler().fit_transform(X)

X_tr, X_tmp, y_tr, y_tmp = train_test_split(X, y, test_size=0.4, random_state=42)
X_cal, X_te, y_cal, y_te = train_test_split(X_tmp, y_tmp, test_size=0.5, random_state=42)

# ── Train MLP ─────────────────────────────────────────────────────
clf = MLPClassifier(hidden_layer_sizes=(128, 64), max_iter=500, random_state=42)
clf.fit(X_tr, y_tr)

# ── Calibration scores: 1 - P(true class) ───────────────────────
cal_probs  = clf.predict_proba(X_cal)
cal_scores = 1 - cal_probs[np.arange(len(y_cal)), y_cal]

def get_qhat(scores, alpha):
    n = len(scores)
    lev = np.ceil((n+1)*(1-alpha)) / n
    return np.quantile(scores, min(lev, 1.0), interpolation='higher')

alpha  = 0.10
q_hat = get_qhat(cal_scores, alpha)

# ── Prediction sets ──────────────────────────────────────────────
te_probs  = clf.predict_proba(X_te)
pred_sets = [np.where(1 - p <= q_hat)[0] for p in te_probs]

coverage  = np.mean([y_te[i] in pred_sets[i] for i in range(len(y_te))])
avg_size  = np.mean([len(s) for s in pred_sets])
empty_pct = np.mean([len(s) == 0 for s in pred_sets])

print(f"q̂         = {q_hat:.4f}")
print(f"Coverage  = {coverage:.3f}  (target ≥ {1-alpha:.2f})")
print(f"Avg size  = {avg_size:.2f}")
print(f"Empty set = {empty_pct:.3f}")
Output

Coverage = 0.913 (target ≥ 0.90)
Avg size = 1.62
Empty set = 0.000

Section 04

RAPS: Regularised Adaptive Prediction Sets

Softmax CP can produce large sets because rare classes can still have scores above threshold. RAPS (Angelopoulos et al., 2021) penalises sets that include many low-probability classes.

s_RAPS(x, y) = Σ_{k: π_k ≤ π_y} f̂(x)[π_k] + λ · (o(y, f̂) − k_reg)₊

Here π_k is the rank of class k by decreasing probability, o(y,f̂) is the rank of the true class, and λ, k_reg are regularisation hyperparameters.

raps_score.py
def raps_scores(probs, labels, lam=0.01, k_reg=2):
    """Compute RAPS nonconformity scores."""
    n, K = probs.shape
    scores = np.zeros(n)
    for i in range(n):
        pi = np.argsort(probs[i])[::-1]   # sorted by descending prob
        rank_y = np.where(pi == labels[i])[0][0]  # rank of true class
        # cumsum up to and including true class rank
        cum = np.cumsum(probs[i][pi])
        s   = cum[rank_y]
        # regularisation: penalise if rank_y > k_reg
        s  += lam * max(rank_y + 1 - k_reg, 0)
        scores[i] = s
    return scores

cal_raps = raps_scores(cal_probs, y_cal)
q_raps   = get_qhat(cal_raps, alpha)

def raps_predict_set(probs, q, lam=0.01, k_reg=2):
    pi  = np.argsort(probs)[::-1]
    cum = np.cumsum(probs[pi])
    reg = np.array([lam * max(k + 1 - k_reg, 0) for k in range(len(pi))])
    return pi[cum + reg <= q].tolist()

raps_sets = [raps_predict_set(p, q_raps) for p in te_probs]
cov_raps  = np.mean([y_te[i] in raps_sets[i] for i in range(len(y_te))])
sz_raps   = np.mean([len(s) for s in raps_sets])
print(f"RAPS coverage : {cov_raps:.3f}")
print(f"RAPS avg size : {sz_raps:.2f}  (Basic: {avg_size:.2f})")
Output
RAPS coverage : 0.908
RAPS avg size : 1.41 (Basic: 1.62)
Section 05

Mondrian CP: Class-Conditional Coverage

Standard conformal gives marginal coverage: P(Y ∈ C(X)) ≥ 1−α averaged over the test distribution. For imbalanced data, rare classes may be under-covered. Mondrian CP computes a separate threshold q̂_k for each class k.

mondrian_cp.py
# ── Class-conditional thresholds ─────────────────────────────────
K      = len(np.unique(y_cal))
q_mond = {}  # one threshold per class
for k in range(K):
    idx_k   = (y_cal == k)
    scores_k = cal_scores[idx_k]
    if len(scores_k) > 0:
        q_mond[k] = get_qhat(scores_k, alpha)

# ── Mondrian prediction set: test each class with its own q̂_k ────
def mondrian_predict(probs, q_mond):
    pset = []
    for k, q_k in q_mond.items():
        if 1 - probs[k] <= q_k:   # class k's own threshold
            pset.append(k)
    return pset

mond_sets = [mondrian_predict(p, q_mond) for p in te_probs]

# ── Per-class coverage ───────────────────────────────────────────
for k in range(K):
    idx = (y_te == k)
    cov_k = np.mean([y_te[i] in mond_sets[i] for i in np.where(idx)[0]])
    print(f"Class {k}: coverage = {cov_k:.3f}")

Section 07

Coverage Guarantee

The same exchangeability argument from regression applies here. The key formula:

P( Y_{n+1} ∈ C_α(X_{n+1}) ) ≥ 1 − α

For classification: C_α(x) = {k : 1 − f̂(x)[k] ≤ q̂}, so the event Y ∈ C_α(X) is equivalent to s(X, Y) ≤ q̂ — exactly the regression guarantee applied to the softmax score.

📐
Marginal vs. conditional coverage

Standard conformal gives marginal coverage: coverage holds on average over (X, Y). Mondrian CP achieves conditional coverage per class. True covariate-conditional coverage (P(Y ∈ C(X) | X=x) ≥ 1−α for all x) requires additional assumptions and is generally impossible without them.

Section 08

Pitfalls and Best Practices

⚠️
Pitfall 1: Overconfident base model → large sets

If the model's softmax scores are poorly calibrated (consistently too high), many near-threshold classes will be included. Temperature scaling before conformal prediction reduces average set size without breaking guarantees.

⚠️
Pitfall 2: Empty prediction sets

When α > 0.5 and the model is very uncertain, sets can be empty — the algorithm finds no class likely enough. This is mathematically valid but practically useless. Clip α to reasonable values (≤ 0.2).

✅
Tip 1: Use RAPS over basic softmax CP

RAPS produces smaller prediction sets with the same coverage guarantee. Prefer RAPS for multi-class problems (K > 10), where softmax CP can produce large sets for uncertain examples.

✅
Tip 2: Use Mondrian CP for imbalanced datasets

If your class distribution is imbalanced, marginal coverage can mask poor per-class coverage on rare classes. Mondrian CP with per-class thresholds is strongly recommended.
