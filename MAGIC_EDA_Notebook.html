§ 01

Dataset Overview & Feature Descriptions

The MAGIC (Major Atmospheric Gamma Imaging Cherenkov) telescope records images of particle showers triggered by high-energy particles entering the atmosphere. Each event is characterized using Hillas parameters — geometric moments of the elliptical camera image. The goal is to distinguish γ-ray photon events (signal) from the far more common hadronic cosmic-ray showers (background).

hadron = df[df['class']==1]

In [12]
def iqr_outliers(col):
    Q1 = df[col].quantile(0.25)
    Q3 = df[col].quantile(0.75)
    IQR = Q3 - Q1
    return df[(df[col]<Q1-1.5*IQR)|(df[col]>Q3+1.5*IQR)]

for col in feats:
    outs = iqr_outliers(col)
    print(f"{col:10s}: {len(outs):4d} outliers ({len(outs)/len(df)*100:.1f}%)")

Empirical Cumulative Distribution Functions (ECDF)

ECDFs show the full distribution without binning artifacts. The horizontal gap between gamma/hadron ECDF curves shows separation power — a large gap means the feature is discriminating.

In [21]
def ecdf(data):
    x = np.sort(data)
    y = np.arange(1, len(x)+1) / len(x)
    return x, y

fig, axes = plt.subplots(2,5, figsize=(18,6))

    ax.step(*ecdf(hadron[col]), color='#f97316', where='post')

from sklearn.metrics import roc_auc_score

aucs = {}
for col in feats:
    aucs[col] = roc_auc_score(df['class'], df[col])
    # AUC > 0.5 means feature correlates with hadron label
    # flip if AUC < 0.5 (feature inversely discriminates)
    if aucs[col] < 0.5:
        aucs[col] = 1 - aucs[col]

✅ Data Quality

• No missing values or duplicates
• No categorical encoding needed
• All features continuous float64
• fAlpha bounded [0°, 90°] — no outliers
• fConc/fConc1 bounded [0, 1]

🔧 Preprocessing

• Use RobustScaler (heavy tails present)
• Log-transform fLength, fWidth
• Consider dropping fConc1 (r=0.99 w/ fConc)
• Consider dropping fM3Trans (AUC=0.508)
• Class-weight or SMOTE for 65/35 imbalance

📐 Feature Engineering

• fAlpha² might improve non-linear models
• Aspect ratio: fLength/fWidth is a derived feature
• Distance-corrected concentration: fConc/fDist
• Sign of fAsym carries info (peak position)
• Interaction: fAlpha × fConc very discriminating

🤖 Model Recommendations

• Random Forest / XGBoost handle skew well
• LDA benefits from Gaussianizing transforms
• SVM with RBF kernel suits this feature space
• Evaluate with AUC-ROC (not accuracy)
• Use stratified 5-fold CV throughout
